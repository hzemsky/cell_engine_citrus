# cell engine citrus

|=============================|
| z3msky's cell engine citrus |
|=============================|

cell engine citrus is a cellular automata toy for the Nintendo 3DS.

For use with homebrew enabled 3DS systems
To use on Citra 3DS emulator, select the .3dsx file.
